# DHT11 circuit wiring

VCC+   = Rouge  -> Blanc / Vert    -> Rouge -> Raspberry Pi Pin 4 (5V)<br/>
Data   = Orange -> Blanc / Orange  -> Noir  -> Raspberry Pi Pin 37 (GPIO 25)<br/>
Ground = Blanc  -> Blanc / Bleu    -> Blanc -> Raspberry Pi Pin 38 (Ground)<br/>
