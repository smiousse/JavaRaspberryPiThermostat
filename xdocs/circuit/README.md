# DHT11 circuit wiring

VCC+   = Rouge  -> Blanc / Vert    -> Rouge -> Raspberry Pi Pin 4 (5V)
Data   = Orange -> Blanc / Orange  -> Noir  -> Raspberry Pi Pin 37 (GPIO 25)
Ground = Blanc  -> Blanc / Bleu    -> Blanc -> Raspberry Pi Pin 38 (Ground)
